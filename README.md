### 開発フェーズ
以下のフェーズに則って開発をしていく

**第一フェーズ**

ざっくりと駒を動かしたり置いたりできる状態

* 簡易的な盤面の表示
* 駒の移動
* 駒の取得
* 駒を置く
* 成り

**第二フェーズ**

将棋の基本的なルールを網羅した状態

* 詰み判定
* 王手判定
* 二歩判定

**第三フェーズ**

複数の対局を管理できる状態

* マッチング

**第四フェーズ**

UIが改善された状態

* TOPページの作成
* マッチングページの作成
* 駒・盤面の改善
* 対戦結果ページの作成
